# Workflow

Use this folder as a shared handoff zone.

## Rules

1. Read `handoff/START-HERE.md` first.
2. Work on a branch.
3. Keep changes small.
4. Update handoff files when context changes.
5. Do not edit the same files from multiple tools at once.
6. Prefer pull requests for larger changes.

## Task split

Live editor:

- Firmware loops.
- Dashboard editing.
- OpenSCAD.
- KiCad-related scripts and files.

Background agent:

- Repo cleanup.
- Documentation.
- Test harnesses.
- CI.
- Reports.

Review agent:

- Review.
- Critique.
- Explain alternate approaches.
- Sanity-check assumptions.
