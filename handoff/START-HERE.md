# AI Handoff Folder

This `handoff/` folder is a temporary shared context folder for ChatGPT, Codex, Claude Code, Cursor, and future World Famous Electronics team members.

Read this first before doing any work.

## Purpose

This folder captures the current working memory for PulseSensor / Big Eye / World Famous Electronics so another AI system or teammate can pick up the project without starting from zero.

This is not the final corporate brain. It is a lightweight handoff cache.

## Instructions for any AI agent

1. Read every file in `handoff/` before proposing changes.
2. Do not overwrite design intent just because a generic engineering pattern looks cleaner.
3. Preserve the educational, maker-friendly, Arduino-friendly spirit of PulseSensor.
4. If you make progress, update `handoff/ACTIVE-TASKS.md`.
5. If you make a meaningful decision, update `handoff/DECISIONS.md`.
6. If uncertain, write the uncertainty down instead of pretending it is resolved.
